## How to run Hello World! on HTML
1. Copy and paste the code below
```
<!DOCTYPE html>
<html>
<body>
<h1>Hello World!</h1>
</body>
</html> 
```
2. Open any Browser you have (Google, Coc Coc, Microsoft Edge, Firefox,...)
3. Using this syntax to run and show the results Hello World! on the site
```
file:///file_path
```
